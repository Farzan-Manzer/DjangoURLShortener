# Running the source
Clone or download the repository zip from 

```
cd DjangoURLShortener
virtualenv venv
. venv/bin/activate
pip install -r requirements.txt
./manage.py migrate
./manage.py createsuperuser
./manage.py runserver
```

Open http://localhost:8000/admin/
